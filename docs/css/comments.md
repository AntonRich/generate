# Comments



## Single Line


```css
/* This is a CSS comment */
div {
   color: red; /* This is a CSS comment */ 
}

```



## Multiple Line


```css
/*
    This
    is
    a
    CSS
    comment
*/
div {
   color: red;
}

```



#### Syntax


- /* Comment */



#### Remarks


- Comments in CSS always start with `/*` and end with `*/`
- Comments cannot be nested
